Практика по лекции №2

Задача №1.


Сделайте проверку существования языка КС грамматики.

E → E+T|T
T → T*F|F
F → (E)|a

Решение


N0 = Ø, i ≠ 1
N1 = Ø ∪ {F} = {F}
N1 ≠ N0, то i = i + 1 = 2
N2 = {F} ∪ {T} = {F, T}
N2 ≠ N1, i = 3
N3 = {F, T} ∪ {E} = {F, T, E}
N3 ≠ N2, i = 4
N4 = {F, T, E}
N4 = N3 = Ne

Задача №2.


Устраните недостижимые символы из данного набора правил

E → E+T|T
T → T*F|F
F → (E)|a

Решение


V0 = {E}, i = 1
V1 = {E} ∪ {+, T}
V1 ≠ V0, i = 2
V2 = {E, +, T} ∪ {*, F}
V2 ≠ V1, i = 3
V3 = {E, +, T, *, F} ∪ {(, ), a}
V3 = V2, i = 4
V4 = V3 = Ve
P' = P

Задача №3.


Проверка существования языка КС-грамматики со следующими правилами:

S → D0G | G
G → G1M
R → 1N0 | ß

Решение


НЕ СУЩЕСТВУЕТ.
Пояснение: Нетерминал β мы можем получить только из терминала R, который является недостижимым символом, из-за чего наши правила не образуют ни одной нетерминальной цепочки из-за чего можно сказать, что такая грамматика не существует

Задача №6.


Удаление бесполезных символов и преобразование в эквивалентную КС-грамматику для следующих правил:
S → α | A
A → AB
B → ß

Решение


Пояснение: Терминалы S и B порождают две нетерминальных цепочки и являются «хорошими» символами, но так как терминал B является недостижимым, то его правила нам придётся отбросить. Терминал A не образует нетерминальных цепочек из-за чего все правила с его участием нам также не понадобятся.

Решение


Ответ:
S → KN
K → αß
N → ß
Пояснение: Нетерминалы A и B являются ε-правилами из-за чего их нужно убрать из правил, а также убрать эти символы из правых частей остальных правил и удалить терминалы, которые состоят ТОЛЬКО из этих ε-правил (и их упоминания в правых частях)

Задача №14.


Удаление цепных правил и преобразование в эквивалентную КС-грамматику для следующих правил:
A → B | αß
B → α | C
C → ß

Решение


Ответ:
A → α | ß | αß
Пояснение: берём нетерминалы и терминалы, из которых они получены. Если в этот терминал можно попасть из другого терминала, то мы берём второй терминал и заменяем в его правилах терминал (образующий нетерминальную цепочку) на соответствующий ему нетерминал.

Практика по лекции №6

Пример 1.

Минимизируйте представленный ниже автомат.

Автомат
Вершины 0 1
→ q0 q1 q2
q1 q2 q4*
q2 q4* q2
q3* q4* q3*
q4* q4* q3
q5 q3* q4*

Решение


Удалим бесполезные вершины. Очевидно здесь, что бесполезная вершина - q5, так как в неё не входит ни одна вершина.


С помощью классов эквивалентности преобразуем наши вершины, объединив в общие классы, а затем объединив их.


C1 = {q0, q1, q2}, C2 = {q3, q4}


q0,0 = q1      q0,1 = q2
q1,0 = q2      q1,1 = q4
q2,0 = q4      q2,1 = q2


q3,0 = q4      q3,1 = q3
q4,0 = q4      q4,1 = q3


C1 = {q0}, C1,2 = {q1, q2}, C2 = {q3, q4}


q1,0 = q2      q1,1 = q4
q2,0 = q4      q2,1 = q2


C1 = {q0}, C1,2 = {q1}, C1,2,1 = q2, C2 = {q3, q4}


Рез. автомат
Вершины 0 1
→ q0 q1 q2
q1 q2 q3*
q2 q3* q1
q3* q3* q3*

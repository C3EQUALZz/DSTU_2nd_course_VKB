Практика по лекции №1

Пример 1.


Для заданных языков L1 и L2 найти L1 ∩ L2 и L1 ∪ L2:
- L1 = {bia | i ≥ 0} i , L2 = {aib | i ≥ 0};
- L1 = {bi aj | i ≥ 0, j ≥ 0} ij , L2 = {ai bj | i ≥ 0, j ≥ 0};

Решение


- L1 ∩ L2 = {bi ai + 1b | i ≥ 0}
Пример: ab, baab, bbaaab...
L1 ∪ L2 = {Ø}
Наборы: L1 = {a, ba, bba, bbba, …}, L2 = {b, ab, aab, aaab, …}. Результат пересечения – пустое множество


- L11 ∪ L2 = {bi aj aii bj | i, j ≥ 0}
Пример: λ, ba, ab, baab, bbaaab, baaabb, bbaaaabb…
L1 ∩ L2 = {λ}
Результатом пересечение L1 и L2 может быть только λ, так как единственный элемент, который есть в L1 и L2, это пустой символ, который можно получить при i, j = 0

Пример 2.


Построить контекстно-свободные грамматики для следующих языков:
L = {0i 1j | i = j};
L = {0i 1j | i ≥ j};

Решение


Главный алгоритм при составлении грамматики должен быть таким, чтобы хотя бы одно правило имело такой вид, например:
A → aCa, A → Bav, B → nLb;
Пример для задания:
S → AB
A → 0A | 0 | λ
B → B1 | 1 | λ

Пример 3.


Описать язык, порождаемый грамматикой G = {a, b}, {S, A}, P, S,
где P состоит из правил:

S → aS | bS | aA
A → a

Решение


Левая часть правил состоит из одного нетерминального символа – это либо регулярная грамматика, либо КС.
Правая часть состоит из терминальных и нетерминальных символов, а также имеют конструкцию - “aA”.
Из этого можно сделать вывод, что это регулярная грамматика, причём праволинейная.

Пример 4.


Описать язык, порождаемый грамматикой G = {a, b}, {S, A, B}, P, S,
где P состоит из правил:

S → AB
A → aA | λ
B → Bb | λ

Решение


Левая часть правил состоит из одного нетерминального символа – это либо регулярная грамматика, либо КС.
Правая часть состоит из терминальных и нетерминальных символов. В правиле “S → AB” в правой части находятся два терминала,
а значит это не может быть регулярной грамматикой – это КС грамматика.

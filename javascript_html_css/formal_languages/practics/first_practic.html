Практика по лекции №1

Пример 1. Для заданных языков L1 и L2 найти L1 ∩ L2 и L1 ∪ L2:


- L1 = {bia | i ≥ 0} i , L2 = {aib | i ≥ 0};
- L1 = {bi aj | i ≥ 0, j ≥ 0} ij , L2 = {ai bj | i ≥ 0, j ≥ 0};

Решение


- L1 ∩ L2 = {bi ai + 1b | i ≥ 0}
Пример: ab, baab, bbaaab...
L1 ∪ L2 = {Ø}
Наборы: L1 = {a, ba, bba, bbba, …}, L2 = {b, ab, aab, aaab, …}. Результат пересечения – пустое множество


- L11 ∪ L2 = {bi aj aii bj | i, j ≥ 0}
Пример: λ, ba, ab, baab, bbaaab, baaabb, bbaaaabb…
L1 ∩ L2 = {λ}
Результатом пересечение L1 и L2 может быть только λ, так как единственный элемент, который есть в L1 и L2, это пустой символ, который можно получить при i, j = 0

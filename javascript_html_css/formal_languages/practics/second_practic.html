Практика по лекции №2

Пример 1.


Сделайте проверку существования языка КС грамматики.

E → E+T|T
T → T*F|F
F → (E)|a

Решение


N0 = Ø, i ≠ 1
N1 = Ø ∪ {F} = {F}
N1 ≠ N0, то i = i + 1 = 2
N2 = {F} ∪ {T} = {F, T}
N2 ≠ N1, i = 3
N3 = {F, T} ∪ {E} = {F, T, E}
N3 ≠ N2, i = 4
N4 = {F, T, E}
N4 = N3 = Ne

Пример 2.


Устраните недостижимые символы из данного набора правил

E → E+T|T
T → T*F|F
F → (E)|a

Решение


V0 = {E}, i = 1
V1 = {E} ∪ {+, T}
V1 ≠ V0, i = 2
V2 = {E, +, T} ∪ {*, F}
V2 ≠ V1, i = 3
V3 = {E, +, T, *, F} ∪ {(, ), a}
V3 = V2, i = 4
V4 = V3 = Ve
